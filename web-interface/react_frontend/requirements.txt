# Requirements pour l'installation du projet React Frontend
# Network Management System - Interface Web

## Prérequis système
# - Node.js >= 18.0.0 (recommandé: 20.x LTS)
# - npm >= 9.0.0 ou yarn >= 1.22.0
# - Git

## Instructions d'installation

### 1. Cloner le repository et accéder au dossier
# git clone <url-du-repo>
# cd network-management-system/web-interface/react_frontend

### 2. Installer les dépendances
# npm install
# ou
# yarn install

### 3. Scripts disponibles
# npm run dev       - Lancer le serveur de développement (Vite)
# npm run build     - Construire pour la production
# npm run preview   - Prévisualiser la version de production
# npm run lint      - Lancer ESLint pour vérifier le code

### 5. Configuration de l'environnement
# Créer un fichier .env.local avec les variables d'environnement nécessaires
# VITE_API_URL=http://localhost:8000
# VITE_WS_URL=ws://localhost:8000/ws

### 6. Démarrer le projet
# npm run dev
# L'application sera disponible sur http://localhost:5173

### 7. Notes importantes
# - Le projet utilise Vite comme bundler
# - ESLint est configuré pour le linting
# - Tailwind CSS est utilisé pour les styles
# - Redux Toolkit pour la gestion d'état
# - Ant Design comme bibliothèque de composants UI
# - Tests configurés avec Jest et Cypress
# - Hot reload activé en mode développement

### 8. Ports utilisés
# - 5173: Serveur de développement Vite
# - 4173: Serveur de prévisualisation
